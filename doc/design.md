# 追加仕様
## ゲーム
- 複数並行して立ち上げられる
- 各ゲームはアドレスによって識別される
- ゲームオーナーは1番目のプレイヤーになる
- マスは16*16マス
- STANDBY,OPEN,CLOSEの状態がある
  - STANDBY
    - ゲーム立ち上げからプレイヤーが二人揃うまで
  - OPEN
    - プレイヤーがそろってからCLOSEまで
  - CLOSE
    - ゲームに決着がつくまたはマスが埋まる

## UI
以下のページ・ボタンがある
- ゲーム一覧ページ
  - ゲーム詳細ページ遷移
  - ゲーム立ち上げボタン
- ゲーム詳細ページ
  - ボード・ゲーム状態が表示される（subgraphで集計）
  - STANDBY状態の時、ゲームエントリーボタン表示
  - OPEN状態でエントリー済みで自分のターンの時ボードに石を置ける
